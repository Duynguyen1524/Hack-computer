assembler.py: The Python script for the Hack assembler.


- Place the .asm Hack assembly file in the same directory as assembler.py.

- Open a terminal or command prompt.

- Run the assembler by executing the following command:

	* Python assembler.py <filename>.asm
		ex:Python assembler.py Max.asm

The assembler will create a file with the same name as the .asm file but with a .hack extension; containing the binary code.
